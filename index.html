Nosso primeiro projeto

O Traffic Education é um software de uma auto escola virtual, onde não se é mais nescessario ir á escola física para realizar as aulas e provas que não envolvam o carro fisicamente.

Cadastro

É feito através da coleta dos dados pessoais do aluno como RG, envio de uma foto, número, data de nascimento, etc.

Tecnologia

Será feito utilizando as linguagens php, css, html e javascript.

Custo

O site para receber manutenção e atualizações será coprada uma taxa de serviço sobre isso.
